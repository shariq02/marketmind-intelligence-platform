# MarketMind Intelligence Platform - V1 Data Engineering Foundation

Event-driven financial data pipeline for US equity markets with Kafka streaming, Airflow orchestration, and multi-layer Parquet storage (Bronze/Silver/Gold). Ingests OHLCV data, macroeconomic indicators, and SEC filings with quality gates and replay capability.

**Tech Stack:** Python 3.11, Redpanda/Kafka, Apache Airflow 2.8, PostgreSQL 18, Parquet, Alpaca API, AkShare, EdgarTools
